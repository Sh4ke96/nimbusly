# Nimbusly

**Your family's shared home — all in one place.**

Nimbusly brings the whole family together. Track spending, plan shopping trips, remember birthdays, manage gift ideas, and stay on top of everyone's schedule — without the chaos of group chats and sticky notes.

Everyone in the family gets their own profile and sees the same live data. You always know who added what and when.

---

## Features

- **Family Budget** — track shared expenses in real time, set limits, browse history
- **Shopping Lists** — create and share lists; everyone sees who added each item
- **Gifts** — plan surprises privately so no one spoils them
- **Birthdays** — never miss one again, with reminders and gift ideas
- **Schedule** — a shared family calendar so everyone knows what's happening
- **Family Account** — up to 6 members, each with their own view and permissions

---

## Tech Stack

- Next.js 16 · React 19 · TypeScript
- Tailwind CSS v4
- Supabase (Auth + PostgreSQL)
- Available in Polish and English

---

## Running Locally

```bash
cp .env.local.example .env.local
# fill in your Supabase URL and anon key

yarn install
yarn dev
```

Open [http://localhost:3000](http://localhost:3000).
